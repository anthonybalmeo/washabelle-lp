## How to use

Install Landing Page and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

Landing page served on ` http://localhost:3000`

Exporting HTML. HTML gets outputed to the `/out` folder which you can use to serve.
```bash
npm run export
```
